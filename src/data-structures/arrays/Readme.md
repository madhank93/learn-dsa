### Big O

| Syntax                     | Operation                   | Complexity |
| -------------------------- | --------------------------- | ---------- |
| at(index) or array[index]  | Search                      | O(1)       |
| splice(start, 0, item)     | Insert                      | O(n)       |
| splice(start, deleteCount) | Delete                      | O(n)       |
| push()                     | Add item to end of an array | O(1)       |
